# Humpback Whale Bioacoustic Analysis

## Project Overview

This project analyzes humpback whale vocalizations using bioacoustic methods. Acoustic recordings were processed in Raven Pro and analyzed in R to characterize vocalization patterns and generate statistical visualizations.

## Objectives

- Analyze humpback whale vocalizations.
- Generate and interpret spectrograms.
- Perform statistical analyses of acoustic parameters.
- Visualize results using graphical methods.

## Tools and Software

- Raven Pro
- R
- RStudio

## Repository Contents

- `manuscript.pdf`- Full project report.
- `humpback_whale_script.R` - Data processing and statistical analysis code.
- `figures/`- Graphs generated during the analysis.
- `spectograms/`- Spectograms used for vocalization characterization.
